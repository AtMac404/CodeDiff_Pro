# CodeDiff Pro

#### 一键启动的浏览器端代码 Diff 工具，AI 时代开发者的便携工具

**项目介绍:** 这是一个为了解决“看完 AI 给的代码不知道改了哪”而诞生的敏捷工具。

没有臃肿的体积，没有复杂的依赖。**CodeDiff Pro** 将强大的 Monaco 差异对比引擎搬到了网页端。你只需要拖入旧版本和新版本的文件夹，它就会自动生成带有状态标识（新增/修改/删除）的工程目录树。点击任意文件，立刻呈现高精度的左右对比视图。

### **主要特性：**

* **🌳 智能目录树**：自动对齐路径，跳过无关文件，清爽直观。
* **⚡️ 极速渲染**：借助静态 CDN 与原生前端读取，百个文件秒级解析。
* **🎨 沉浸体验**：内置 VS Dark 暗黑主题，完美还原主流 IDE 观感。
* **🔒 绝对隔离**：跨平台（Win/Mac）一键脚本启动，纯本地运行，断网可用。
